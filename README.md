# Arduino LED Increment and Decrement Counter

## 📌 Project Overview

This project demonstrates an **Increment and Decrement LED counter using Arduino UNO**.
Multiple LEDs are connected to Arduino digital pins and are used to display the count.

When the increment input is triggered, the LED count increases.
When the decrement input is triggered, the LED count decreases.

## 🛠 Components Used

* Arduino UNO
* LEDs (5)
* Push Buttons (Increment & Decrement)
* Resistors
* Proteus Design Suite

## ⚙ Working Principle

* LEDs represent the current count value.
* Pressing the **Increment button** increases the LED count.
* Pressing the **Decrement button** decreases the LED count.
* Arduino processes the button inputs and updates the LEDs accordingly.
